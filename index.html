他山之石，可以攻玉

引用 http://fangzh.top/2018/2018090514/ hexo教程：github page+独立域名搭建(1)

问题1：教程图片显示不全
解决方法：尝试将网页信息复制到第三方编辑软件，比如word可正常显示

问题2：npm install命令卡顿问题
解决方法：尝试修改安装源
$ npm config set registry https://registry.npm.taobao.org //国内镜像
$ npm config get registry //测试配置是否成功

问题3：域名配置完成，未能正常登陆（国内申请域名）
解决方法：实名认证完成后，尝试解析
